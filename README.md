# To Do App

A task management app inspired by Microsoft To Do. I built it using React and Firebase.

## Features

- Sign in with Google or continue as a guest
- Add, delete and complete tasks
- Add steps, notes and due dates to tasks
- Smart lists: My Day, Important, Planned
- Create custom lists
- Search tasks
- List and grid view
- Real-time sync with Firebase

## Technologies

- React 19
- Vite
- Firebase (Firestore + Authentication)
- Tailwind CSS
- Lucide React (icons)
